# umejug-mutation-testing-lab

## OSX  / Linux

#### Mutation testing
```> ./gradlew pitest```

Reports can viewed using _build/reports/index.html_
#### Running the application
```> ./gradlew bootRun```

## Windows

#### Mutation testing
```$ gradlew.bat pitest```

Reports can viewed using _build\reports\index.html_
#### Running the application
```$ gradlew.bat bootRun```
